# Análise Exploratória de Dados - IMDB

Projeto desenvolvido para a disciplina Técnicas e Algoritmos em Ciência de Dados.

## Objetivo

Realizar uma Análise Exploratória de Dados (EDA) utilizando o dataset IMDB.

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Como Executar

1. Clone o repositório
2. Instale as dependências
3. Execute o notebook
